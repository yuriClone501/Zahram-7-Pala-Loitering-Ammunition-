Zahram-7 Loitering Munition

A custom weapon mod for Nuclear Option that adds the Zahram-7, a jet-powered loitering munition for the PALA faction.
Features

- 75kg High-Explosive Warhead – Effective against armored and fortified targets

- 000km Range – Extended operational reach

- 500+ km/h Cruise Speed – Rapid response capability

- Optical seeker with satellite and 5G support

- Air-Launch Capable – Compatible with all aircraft hardpoints

Installation

- Download the latest release

- Extract the contents to BepInEx/plugins/

- Launch the game

- Find the Zahram-7 in your loadout menu

Requirements

- BepInEx 5.x
